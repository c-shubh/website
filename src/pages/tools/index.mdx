# Tools

My implementation of simple tools that I find myself looking up every time I need them.

- [Bulk URL Opener](/tools/bulk-url-opener) - Paste a list of URLs to open them all in new tabs.
- [QR Code Generator](/tools/qr-code-generator) - Quickly generate a QR code and download or copy it to clipboard.
- [Slugify](/tools/slugify) - Create [slug](https://en.wikipedia.org/wiki/Clean_URL#Slug) from text.
- [Pomodoro Timer](/tools/pomodoro-timer) - A simple pomodoro timer.
